---

**Tooka** – _"Your tasks, your way. Organize, manage, and conquer!"_

---

Tooka is a cross-platform, terminal-based task manager that helps you stay on top of your to-do list with clarity and style. It supports adding, editing, listing, and completing tasks — all from a clean and intuitive command-line interface.

Each task can have a title, description, due date, priority, and completion status. Tooka keeps things simple while giving you enough structure to stay focused.

---

## ✨ Features

- Fast and friendly CLI
- Rich terminal UI with colors and layout
- Due date and priority support
- Clear statuses: Pending, Completed, Due Today, Overdue
- Toggle task completion with ease

---

## 📦 Installation

Head over to the [Releases page](https://github.com/Benji377/tooka/releases) and download the binary for your operating system.

No setup, no fuss — just run it:

```bash
./tooka
````

---

## 🚀 Usage

To get started, run:

```bash
./tooka help
```

This will list available commands like `add`, `edit`, `remove`, `list`, and more.

For detailed usage, examples, and shell completion setup, see the [Wiki](https://github.com/Benji377/tooka/wiki).
